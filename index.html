My Projects

Here are some examples of my work

Simon Piano

Github

For this project, I chose to create a 'Simon' type of game on a piano keyboard. This game was inspired by an ear training game I would use with my students when I was a music instructor. The students would play back a melody after first hearing me play it. The technologies I used on this project were Javascript, CSS, HTML, and Git for version control.

I fell in love with this project and had a blast creating it. After getting the basic logic of a 'Simon' game working, I introduced a variety of other features, like binding the piano keys to a computer keyboard, turning the key lights on or off, free play mode, and even a few songs.

Synth Builder

Github

This website lets you build your own synthesizer. Change the pitch, volume, wave form, add effects, and then add as many synths as you would like! I built this site using a React framework. I used ToneJS, which builds off Web Audio API for the sound creation and manipulation. I made this site largely to get more familiar with using React while learning about how to use the Web Audio API and online sound libraries to create interesting sounds and music.

Find My Recipe

Github

Find My Recipe is a group project that I worked on. We decided to create a recipe finder application that would recommend recipes based off of ingredients the user listed. The user enters in their main ingredient they would like to search for, and then can filter the results further by entering in subsequent ingredients. Users can also add in new recipes or edit existing ones. The technologies we used were Ruby on Rails, AngularJS, Javascript, HTML, CSS, Web APIs, MongoDB, Heroku, and Git.

Star Wars Wikipedia

Github

This project is a wikipedia type website specifically for Star Wars information. You can see info and images about characters, species, vehicles, planets, and alliances. You can also edit, and delete any of these models, or create your own! This site was built on Ruby on Rails. I also used two Web APIs (SWAPI for Star Wars Data, and Flickr for images), ActiveRecord, PostgreSQL, CSS, HTML, and Git.
